TITLE
=====

jQuery Plugin: Test Font - version 0.1


DESCRIPTION
===========

Test font availability.


USAGE
=====

Alert if "Helvetica Neue" is available:

    if ($.testFont("Helvetica Neue")) {
      alert("Available!")
    }
